Ayuda - Operaciones con Matrices

Suma de Matrices

Se realiza la suma elemento a elemento. Las dos matrices deben tener las mismas dimensiones.
Ejemplo: Si A = [[1,2],[3,4]] y B = [[5,6],[7,8]], A+B = [[6,8],[10,12]].

Resta de Matrices

Se resta elemento a elemento. Las dos matrices deben tener las mismas dimensiones.
Ejemplo: Si A = [[5,6],[7,8]] y B = [[1,2],[3,4]], A-B = [[4,4],[4,4]].

Multiplicación de Matrices

El número de columnas de la primera matriz debe coincidir con el número de filas de la segunda.
Ejemplo: Si A es 2×3 y B es 3×2, el resultado será una matriz 2×2.

Multiplicación por Escalar

Multiplica cada elemento de la matriz por el valor ingresado.
Ejemplo: Si A = [[1,2],[3,4]] y el escalar es 2, 2A = [[2,4],[6,8]].

Determinante

Calcula el determinante de una matriz cuadrada (solo para matrices de hasta 3×3).
Ejemplo: Para A = [[a,b],[c,d]], det(A) = ad - bc.

Matriz Inversa

Calcula la inversa de una matriz cuadrada (disponible solo para matrices 2×2) siempre que el determinante no sea cero.
Ejemplo: Si A = [[a,b],[c,d]], A⁻¹ = (1/det(A)) * [[d, -b],[-c, a]].

Transpuesta

Intercambia filas por columnas en la matriz.
Ejemplo: La transpuesta de [[1,2],[3,4]] es [[1,3],[2,4]].
